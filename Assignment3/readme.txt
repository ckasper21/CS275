Files:
HW3.html
server_HW3.js

Javascript functions:
- Calculate(): Gets value from text field and dropdown menu, calls request()
- request(): AJAX request to server_H3 Node.js server to obtain
    values for the nth factorial or summation series of n

- Factorial(): calculates nth factorial
- SummationSeries(): calculates the summation series of numbers from 1 to n

How to run:
- Start server_HW3.js
- On web browser, enter address: http://localhost:8080/HW3.html
- Enter in a number in the text field for calculation, make calculation
    selection, and hit Calculate.
