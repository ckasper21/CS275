How to run:
-------------
- Start server_HW4.js
- On web browser, enter address: http://localhost:8080/index.html
- Use the menu in the top left corner to toggle the two other pages
    - Calculator
        - Enter a valid number in the text field
        - Selection desired calculation
        - Hit the calculate button
    - Weather
        - Enter a valid 6 digit zip code
        - Hit the forecast button
