Javascript functions:
- Forecast(): checks for client ID and key, geolocation authentication
- displayForecast(): parses JSON object to be displayed as table
- request(): AJAX request to Aeris Weather to get a 7 day forecast for the users
    current location

How to run:
- Open cdc75_HW2.html with any web browser of your choosing.
- To use the forecast functionality:
    - You must share your location with the browser
    - Enter in a Client ID and Key provided by the Aeris Weather api
